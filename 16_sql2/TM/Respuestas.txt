1. A qué se denomina JOIN en una base datos?

Se denomina JOIN en una BD a combinar varias tablas que manejan atributos en comun esto con el fin de unirlas y poder obtener campos de otras tablas en una misma query.

3. Para qué se utiliza el GROUP BY

    Este comando nos sirve para agrupar registros apartir de una consulta.

4. Para qué se utiliza el HAVING?

    Nos permite realizar filtro parecidos al where pero cuando se agrupan filas, por lo tanto para utilizar el el having se debe utilizar el group by
