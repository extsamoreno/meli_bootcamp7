1. A qué se denomina JOIN en una base datos?
Join es el proceso de tomar datos de varias tablas y colocarlos en una vista generada. 

2. Nombre y explique 2 tipos de JOIN.
FULL JOIN: Es la unión entre dos tablas dada una o más condiciones.
INNER JOIN: Combinación de una fila de una tabla con otra fila de otra tabla.


3. Para qué se utiliza el GROUP BY?
Se utiliza para agrupar los resultados según las columnas indicadas.

4. Para qué se utiliza el HAVING?
Se usa en el GROUP BY para afectar a los grupos seleccionados.
